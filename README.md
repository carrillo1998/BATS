# Sub-daily Bermuda Atlantic Time Series virus sampling reveals taxonomy, host, and functional differences at the population, but not community level

This GitHub contains the scripts for data analysis done on the paper titled [Sub-daily Bermuda Atlantic Time Series virus sampling reveals taxonomy, host, and functional differences at the population, but not community level](https://www.my_paper.com).

## Scripts
The following folders contain scripts for data analysis and figures.

* [Scripts](../master/Scripts/) contains all scripts to reproduce the analyses and figures (main and supplements).

* [Supplemental Files](../master/Supplemental%20Files/) contains the input files. Some of these inputs will be generated through the scripts and have been placed here for ease of access.

## Data availability 
Raw sequence reads and metadata have been submitted to the JGI under [Proposal ID: 505733](https://genome.jgi.doe.gov/portal/Infvirtimeseries/Infvirtimeseries.info.html)

Further data for scripts can be found in a [Zenodo repository](https://doi.org/10.5281/zenodo.17260609)
